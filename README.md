# FutureFinance AI 🚀

## Next-Generation Financial Intelligence Platform

FutureFinance AI is an intelligent financial assistant that bridges the gap between banks' data and customers' financial confusion. Instead of just recommendations, it predicts your financial future.

### Problem Statement

Banks have the data. Customers have the confusion.

- Should I take a loan?
- FD or Mutual Fund?
- Buy phone on EMI or wait?
- When should I buy a home?
- How much emergency fund do I need?

**FutureFinance AI solves this gap through predictive financial intelligence.**

### Key Features

✨ **8 Core Modules:**

1. **Loan Decision AI** - Safe borrowing limits & EMI analysis
2. **Investment Decision AI** - FD vs Mutual Fund vs Gold comparison
3. **Purchase Decision AI** - Affordability & timing recommendations
4. **Retirement Planner** - Long-term wealth projection
5. **Goal Planner** - Achieve dreams with monthly investment
6. **Emergency Risk Detector** - Financial safety alerts
7. **Financial Health Score** - Personalized financial assessment
8. **What-If Simulator ⭐** - Real-time financial scenario testing

### What Makes It Different

❌ **Not Just Recommendations** - Decision Intelligence

**Example:**
```
User: I want to take a ₹10 lakh loan

AI Response:
├─ Current EMI: ₹8,000 (42% of income)
├─ New EMI: ₹21,800
├─ Risk: HIGH
├─ Alternative 1: ₹8 lakh loan → Save ₹80,000 interest
├─ Alternative 2: ₹7 lakh loan → Retire 2 years earlier
└─ Impact: Emergency fund drops to ₹12k (UNSAFE)
```

### Quick Start

#### Prerequisites
- Node.js 16+
- Java 17+
- Python 3.9+
- PostgreSQL 14+
- Docker & Docker Compose

#### Installation

```bash
# Clone repository
git clone https://github.com/nandubodra/FutureFinance-AI.git
cd FutureFinance-AI

# Setup with Docker
docker-compose up -d

# Frontend
cd frontend
npm install
npm start

# Backend
cd ../backend
mvn clean install
mvn spring-boot:run

# ML Engine
cd ../ml-engine
pip install -r requirements.txt
python app.py
```
